AIDEPOR — App Assets
=====================

Replace the following files with your own before building:

  icon.png           — 1024 x 1024 px  — App icon (iOS + Android)
  adaptive-icon.png  — 1024 x 1024 px  — Android adaptive icon foreground
  splash.png         — 1242 x 2436 px  — Splash screen
  favicon.png        —   48 x  48 px   — Web favicon

All images must be PNG format with no alpha channel on icon.png and adaptive-icon.png.

Free tools for generating icons:
  https://www.appicon.co/
  https://icon.kitchen/
  https://makeappicon.com/
